# HolySheet

Prototype de systeme Foundry VTT v14 pour un jeu D100 roll-under simple, generique et customisable.

## Prototype 1

- Manifest `system.json` compatible Foundry VTT v14.
- Types d'Actors `character` et `npc`.
- Type d'Item generique `equipment`.
- Parametres monde internes pour modules de fiche, aptitudes, competences communes, categories, monnaies et etats custom.
- Feuilles Actor et Item basiques.
- Jets `1d100` roll-under depuis les aptitudes et competences.

## Installation

Install from the Foundry setup screen using this system manifest URL:

```text
https://github.com/NathanDigiDev/holysheet-foundry-modules/releases/latest/download/holysheet-system.json
```

## Installation locale

Place ce dossier dans le repertoire `Data/systems/holysheet` de Foundry VTT, puis active le systeme lors de la creation d'un monde.
