# Holysheet Lockpicking

Foundry VTT v14 module for GM-driven lock and minigame interactions, including lockpicking and combination locks.

## Features

- Lockpicking minigame inspired by tactile RPG lock interactions.
- Combination lock minigame using configurable symbols.
- GM sidebar panel for managing minigames and folders.
- Send a minigame to selected connected players.
- Uses sockets so player success can be applied by the GM where required.
- Includes English and French localization.

## Compatibility

- Foundry VTT v14.
- System agnostic.

## Installation

Install from the Foundry setup screen using this manifest URL:

```text
https://github.com/NathanDigiDev/holysheet-foundry-modules/releases/latest/download/holysheet-lockpicking-module.json
```

Then enable **Holysheet Lockpicking** in your world module settings.
