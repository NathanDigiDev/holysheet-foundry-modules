# Holysheet Foundry Friendly Interface

Lightweight Foundry VTT v14 module that adds HolySheet player shortcuts without replacing or reshaping the native Foundry interface.

## Features

- Adds player-friendly access points for character and notes workflows.
- Keeps Foundry's native sidebar and controls intact.
- Uses the HolySheet visual language while staying unobtrusive.
- Includes English and French localization.

## Compatibility

- Foundry VTT v14.
- System agnostic.

## Installation

Install from the Foundry setup screen using this manifest URL:

```text
https://github.com/NathanDigiDev/holysheet-foundry-modules/releases/latest/download/holysheet-foundry-friendly-interface-module.json
```

Then enable **Holysheet Foundry Friendly Interface** in your world module settings.
