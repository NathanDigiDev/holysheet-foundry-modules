# Banner theme backgrounds

Drop optional background images here, referenced by the presets in
`scripts/config.mjs` (`BANNER_PRESETS`):

- `medieval.webp`
- `scifi.webp`
- `steampunk.webp`

These are **optional** — banners render their themed gradient and glow without
them. The per-quest "Banner Background" field in the editor overrides the preset.
