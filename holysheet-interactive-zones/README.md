# Holysheet Interactive Zones

System-agnostic Foundry VTT v14 module for creating scene-stored interactive zones that players can click or inspect during play.

## Features

- Create interactive zones directly on scenes.
- Highlight areas, trigger chat messages, open content, play sounds, run macros, or change scenes.
- Supports voting and player-facing interaction flows.
- Stores zone data on the scene so it travels with the scene.
- Includes English and French localization.

## Compatibility

- Foundry VTT v14.
- System agnostic.

## Installation

Install from the Foundry setup screen using this manifest URL:

```text
https://github.com/NathanDigiDev/holysheet-foundry-modules/releases/latest/download/holysheet-interactive-zones-module.json
```

Then enable **Holysheet Interactive Zones** in your world module settings.
